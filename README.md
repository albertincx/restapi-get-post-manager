# REST API GET/POST Manager

A web-based tool for managing and testing REST API GET and POST requests.

## Description

This application provides a user-friendly interface for sending GET and POST requests to REST APIs. It allows developers and testers to easily test API endpoints, inspect responses, and manage request parameters.

## Features

- Send GET requests to API endpoints
- Send POST requests with customizable payloads
- View and inspect API responses
- Manage request headers
- Save and organize frequently used API endpoints
- Simple and intuitive web interface

## Technologies Used

- HTML
- CSS
- JavaScript
- REST API protocols

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/restapi-get-post-manager.git
   ```

2. Navigate to the project directory:
   ```bash
   cd restapi-get-post-manager
   ```

3. Open `index.html` in your preferred web browser:
   ```bash
   open index.html
   ```

## Usage

1. Enter the API endpoint URL in the designated field
2. Select the HTTP method (GET or POST)
3. Add any required headers or request body (for POST requests)
4. Click the "Send Request" button
5. View the response in the results panel
